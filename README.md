# Group6_BloodSugarMonitoring

Blood Sugar Monitoring System - AI-Powered Health Monitoring Platform

## Project Structure

- `Backend/` - Backend API and server logic
- `src/` - Frontend Next.js application
- `config/` - Configuration files

## Frontend (Next.js)

This is a [Next.js](https://nextjs.org) project for the Blood Sugar Monitoring frontend.

### Getting Started

First, install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm start` - Start production server
- `npm run lint` - Run ESLint

## Backend

See the `Backend/` directory for backend documentation and setup instructions.
